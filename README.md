# Vue ToDo List

Простой ToDo List, созданный на Vue.js.

## Демо

Демо нет

## Функциональность

*   Добавление задач
*   Удаление задач
*   Ввод заголовка и описания для каждой задачи

## Технологии

*   Vue.js
*   HTML
*   CSS

## Установка
Клонируйте репозиторий:
   ```bash
   git clone https://github.com/smaulik17/vue-todo-list.git
   cd vue-todo-list
   npm install
   npm run serve
